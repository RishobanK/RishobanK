## 🔥 Featured Projects

### 🛡️ [Secure STM32 Bootloader & Firmware Update System](https://github.com/RishobanK/secure-stm32-bootloader)

Custom STM32 Blue Pill bootloader focused on ARM Cortex-M firmware validation and secure update concepts.  
The project implements relocated application execution, flash erase/write handling, application header validation, update flag mechanism, CRC32 integrity checking, and bootloader-to-application jump control.

**Focus:** Bootloader design, firmware validation, flash memory layout, CRC32 integrity checking, ARM Cortex-M boot flow  
**Tech:** Embedded C, STM32 Blue Pill, STM32CubeIDE, ST-Link, Linker Script, CRC32, Python  

---

### 🚗 [AGV Embedded Control System](https://github.com/RishobanK/agv-embedded-control)

Embedded firmware for an industrial Autonomous Guided Vehicle used for material transport, with PID-based line following, calibrated load-cell weighing, adaptive white-zone turning, manual override, and non-blocking real-time control.

**Focus:** Real-time motion control, sensor integration, PID line following, EEPROM calibration storage  
**Tech:** Embedded C/C++, ATmega2560, IR Sensor Array, Load Cell, OLED, Stepper Motors  

---

### 🧩 [ESP32-S3 Custom Development Board](https://github.com/RishobanK/esp32-s3-custom-dev-board)

Custom ESP32-S3 development board project for embedded prototyping, including power regulation, boot/reset circuitry, GPIO breakout, programming interface, and sensor/IO expansion.

**Focus:** PCB design, schematic design, embedded hardware prototyping, GPIO/UART/I2C/SPI expansion  
**Tech:** ESP32-S3, KiCad, PCB Design, Power Regulation, USB-C, GPIO

### 🧠 [Wearable Kyphosis Management System](https://github.com/RishobanK/wearable-kyphosis-management-system)

Wearable embedded system for posture-related monitoring using EMG and gyroscope sensors, with real-time feedback using vibration motors.

**Focus:** Sensor acquisition, signal filtering, posture feedback, wearable electronics
